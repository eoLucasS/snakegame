## 📝 Descrição 

O snake game é um clássico dos jogos de arcade, onde você controla uma cobra e deve comer maçãs para aumentar seu tamanho. Desenvolvido usando HTML, CSS e JavaScript, o jogo proporciona uma experiência divertida e desafiadora.

## 🎮 Como Jogar

Para começar a jogar, clique em qualquer tecla do teclado para iniciar o jogo. Caso não pressione nenhuma tecla, o jogo não será iniciado. Divirta-se jogando o Jogo da Cobrinha!

- Preview: [https://snakegame-lucas.vercel.app](https://snakegame-lucas.vercel.app)
